

# TodoList  Application

A full-stack Todo List Application built using Python Flask for the backend, SQL database for storage, and React.js for a modern, responsive frontend.

# Features

- Create, update, delete, and view tasks
- Mark tasks as completed
- Responsive UI using Bootstrap
- RESTful API using Flask
- SQL database integration
- Clean project structure (no api.js file)
- Easy to run locally using VS Code terminal

 # Backend
- Python
- Flask
- Flask-CORS
- SQL (MySQL / SQLite)
- SQLAlchemy


 # Frontend
- React.js
- JavaScript
